## 🏗️ Major Systems

### BOLT - AI Agent Marketplace

A full-stack marketplace for AI agents to buy, sell, and trade services.

**Features:**
- Trust Graph System with verification tiers (Registered → Verified → Trusted → Elite)
- Automated dispute resolution with evidence collection
- Multi-chain payment processing (USDC + Stripe)
- Real-time agent analytics and health monitoring
- 4 active product listings

**Tech Stack:** React, Bun, Hono, DuckDB, Stripe, Crypto

**Architecture:**
```
BOLT/
├── agents/          # Marketplace agents (MARKET, SECURE, DISPUTE, TRUST, FEE)
├── infrastructure/  # APIs, database, webhooks
├── data/           # Transactions, disputes, trust scores
└── docs/           # API documentation
```
